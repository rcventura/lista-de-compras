- Working on `c:\Projetos\Flutter\lista_compras`.

## What this agent does
1. Reads code and project files to locate the app entry and current behavior.
2. Suggests minimal, incrementally safe code changes (hot reload friendly).
3. Applies conventions for Flutter 3+ null safety and standard widget patterns.
4. Adds or updates tests under `test/` when asked.
5. If requested, scaffolds new pages/widgets using existing style.
